# ThoughtLoom : Social Media App
- ThoughtLoom is A platform where ideas are woven into rich text posts with secure auth and image uploads.
- Implemented secure user authentication, including signup, login, and logout, leveraging Appwrite’s robust backend
services. Utilized Appwrite’s database and storage buckets for efficient data management and file handling.
- Employed Redux Toolkit for state management, ensuring smooth data flow for authentication and post functionalities.
Configured environment variables to enable secure, scalable, and maintainable project setup.
- Designed and integrated a content management system allowing users to create, edit, delete, and display posts with
image uploads and rich text formatting using TinyMCE.

## Tech Stack 
### frontend 
- HTML
- CSS
- JavaScript
- ReactJS
- Redux Toolkit
### Backend ( use of BAAS )
- Appwrite
### Other Library 
- React Hook Form
- TinyMCE
